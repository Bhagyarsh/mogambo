# mogambo
## steps

- mkdir mogambo
- cd mogambo
- virtualenv -p python3.6 .
- git clone https://github.com/Bhagyarsh/mogambo.git
- cd mogambo
- source ../bin/activate
- cd mogambo/
- pip install -r requirements.txt
- python manage.py runserver

- cd frontend/
- npm install angular
- sudo npm install -g bower
- ng serve
